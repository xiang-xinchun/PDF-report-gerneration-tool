# 快速开始指南

## 重构完成 ✅

项目已成功重构为标准的 Electron 项目结构。

## 目录结构

```
course-report-generator/
├── src/
│   ├── main/              # 主进程代码
│   ├── renderer/          # 渲染进程代码
│   │   ├── scripts/       # JavaScript 文件
│   │   └── styles/        # CSS 样式
│   └── preload/           # 预加载脚本
├── assets/images/         # 静态资源
├── examples/              # 示例文件
└── package.json
```

## 运行项目

### 1. 安装依赖（首次运行）
```bash
npm install
```

### 2. 启动应用
```bash
npm start
```
或双击运行：`启动应用.bat`

### 3. 打包应用
```bash
npm run build:win
```
或双击运行：`windows打包.bat`

## 问题排查

如果遇到问题：
1. 检查 Node.js 是否已安装（推荐 v16+）
2. 重新安装依赖：`npm install`
3. 清理并重新打包：`npm run clean && npm run build:win`

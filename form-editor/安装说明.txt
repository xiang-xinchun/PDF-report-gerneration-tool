================================================================
课程目标达成情况评价报告生成工具 - 安装说明
================================================================

【三种版本选择】

1. 安装程序版 (Setup.exe)
   - 文件名: 课程目标达成情况评价报告生成工具-1.0.0-x64.exe
   - 需要: .NET Framework 4.5 或更高版本
   - 特点: 传统安装方式，会在开始菜单和桌面创建快捷方式
   - 推荐: Windows 10/11 用户

2. 便携版 (Portable.exe) ★推荐 Windows 7 用户★
   - 文件名: 课程目标达成情况评价报告生成工具-1.0.0-x64-Portable.exe
   - 需要: 无特殊要求（系统通常已自带VC++运行时）
   - 特点: 双击即可运行，无需安装
   - 优点: 
     * 不需要 .NET Framework
     * 不需要管理员权限
     * 可在U盘中运行
   - 推荐: Windows 7 用户首选

3. 压缩包版 (.zip)
   - 文件名: 课程目标达成情况评价报告生成工具-1.0.0-x64.zip
   - 需要: 无特殊要求
   - 特点: 解压后运行，完全便携
   - 优点: 与便携版相同，但更小巧
   - 推荐: 需要在多台电脑使用

================================================================
【Windows 7 用户指南】★重点★
================================================================

推荐使用：便携版 (Portable.exe) 或 压缩包版 (.zip)

步骤1: 选择版本
   - 下载 "xxx-Portable.exe" 或 "xxx.zip"

步骤2: 运行
   - Portable.exe: 双击直接运行
   - .zip: 解压到任意文件夹，双击里面的.exe文件

步骤3: 如果提示缺少运行库
   - 下载并安装: Visual C++ Redistributable 2015-2022
   - 64位系统: https://aka.ms/vs/17/release/vc_redist.x64.exe
   - 32位系统: https://aka.ms/vs/17/release/vc_redist.x86.exe
   - 注意: 大多数Windows 7已安装此运行库

步骤4: 可选 - 创建桌面快捷方式
   - 右键点击 .exe 文件 → 发送到 → 桌面快捷方式

【优势】
✓ 无需 .NET Framework
✓ 无需管理员权限
✓ 双击即可运行
✓ 可在U盘运行
✓ 不写注册表

================================================================
【Windows 10/11 用户指南】
================================================================

推荐使用：安装程序版 (Setup.exe)

步骤1: 下载
   - 下载 "xxx-Setup.exe"

步骤2: 安装

   - 如提示需要.NET Framework，系统会自动引导安装
   - 选择安装目录
   - 完成安装

步骤3: 运行
   - 双击桌面图标或从开始菜单启动

【优势】
✓ 自动创建快捷方式
✓ 集成到系统
✓ 支持自动更新（未来版本）

【64位 vs 32位选择】
================================================================

不知道系统类型？
右键点击"计算机" → 属性 → 查看"系统类型"

- 显示 "64位操作系统" → 下载 x64 版本
- 显示 "32位操作系统" → 下载 ia32 版本

建议：如无特殊情况，现代电脑均使用 x64 版本

【常见问题】
================================================================

Q: 便携版和压缩包版有什么区别？
A: 功能完全相同。便携版是单个exe文件，压缩包需要先解压。

Q: 便携版的配置保存在哪里？
A: 保存在 %APPDATA%\course-report-generator

Q: 可以同时安装多个版本吗？
A: 可以。便携版和压缩包可以共存，但不建议同时安装安装程序版。

Q: Windows 7必须是SP1吗？
A: 是的。Electron 21需要Windows 7 SP1或更高版本。

Q: 还是提示缺少DLL文件怎么办？
A: 安装 Visual C++ Redistributable 2015-2022
   下载地址已在上方说明中提供。

Q: 为什么推荐Win7用户使用便携版？
A: 因为便携版不需要.NET Framework，减少了依赖，更容易在老系统上运行。
